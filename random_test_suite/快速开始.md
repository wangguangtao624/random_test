# CIS芯片TRNG随机性测试套件 - 快速开始

## 1. 安装依赖

```bash
pip install numpy scipy matplotlib
```

或运行安装脚本：
```bash
install.bat
```

## 2. 启动GUI测试平台（推荐）

```bash
python gui_app.py
```

或双击运行：
```bash
启动测试平台.bat
```

**GUI平台功能：**
- 可视化选择测试项目
- 查看每个测试项的详细说明
- 选择测试数据文件或目录
- 实时查看测试进度和日志
- 生成测试报告

## 3. 命令行方式（高级用户）

### 测试单个文件
```bash
python main.py --file data/sample.bin
```

### 测试整个目录
```bash
python main.py --data-dir ./data
```

### 指定输出目录和格式
```bash
python main.py --data-dir ./data --output ./reports --format html
```

## 4. 查看报告

测试完成后，在 `reports/` 目录下会生成：
- `report_YYYYMMDD_HHMMSS.txt` - 文本报告
- `report_YYYYMMDD_HHMMSS.html` - HTML报告（含图表）
- `results_YYYYMMDD_HHMMSS.json` - 详细结果数据

## 5. 测试覆盖

本工具覆盖两个标准的全部测试项：

### NIST SP 800-22 (15项)
1. Frequency (Monobit) Test - 单比特频数检测
2. Frequency Test within a Block - 块内频数检测
3. Runs Test - 游程检测
4. Test for Longest Run of Ones - 块内最长游程检测
5. Binary Matrix Rank Test - 二元矩阵秩检测
6. Discrete Fourier Transform Test - 离散傅里叶检测
7. Non-overlapping Template Matching Test - 非重叠模板匹配检测
8. Overlapping Template Matching Test - 重叠模板匹配检测
9. Maurer's Universal Statistical Test - Maurer通用统计检测
10. Linear Complexity Test - 线性复杂度检测
11. Serial Test - 序列检测
12. Approximate Entropy Test - 近似熵检测
13. Cumulative Sums Test - 累加和检测
14. Random Excursions Test - 随机游动检测
15. Random Excursions Variant Test - 随机游动变体检测

### GB/T 32915-2016 补充测试 (3项)
16. Poker Test (m=4) - 扑克检测
17. Poker Test (m=8) - 扑克检测
18. Autocorrelation Test - 自相关检测
19. Binary Derivative Test - 二元推导检测

## 6. 通过标准

| 指标 | 要求 |
|------|------|
| 单项测试P-value | ≥ 0.01 |
| 总体通过率 | ≥ 98% |
| 测试序列长度 | ≥ 1Mbit/组 |
| 测试序列组数 | ≥ 1000组 |

## 7. 快速验证

```bash
python generate_sample_data.py --output ./data --count 10 --length 1000000
```

## 9. 常见问题

### Q: 测试失败怎么办？
A: 检查数据质量，确保数据采集正确，长度足够。

### Q: 内存不足怎么办？
A: 减少同时处理的文件数量，或使用生成器模式。

### Q: 如何自定义测试参数？
A: 编辑 `config.py` 文件修改参数。

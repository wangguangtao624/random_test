## 文件说明

### 主程序文件

#### `main.py`
主程序入口，提供命令行接口，支持：
- 测试单个文件
- 测试整个目录
- 生成多种格式报告

**使用方法：**
```bash
python main.py --file data/sample.bin
python main.py --data-dir ./data
python main.py --data-dir ./data --output ./reports --format html
```

#### `test_tool.py`
工具验证脚本，用于：
- 验证NIST测试套件功能
- 验证GB/T测试套件功能
- 验证数据加载器功能

**使用方法：**
```bash
python test_tool.py
```

#### `quick_test.py`
快速测试脚本，用于：
- 快速验证工具是否正常工作
- 生成随机测试数据
- 运行基本测试

**使用方法：**
```bash
python quick_test.py
```

#### `generate_sample_data.py`
示例数据生成脚本，用于：
- 生成测试用的随机数据
- 支持多种数据类型（随机、有偏、周期性）
- 支持多种文件格式

**使用方法：**
```bash
python generate_sample_data.py --output ./data --count 10 --length 1000000
```

### 核心测试模块

#### `core/nist_tests.py`
NIST SP 800-22 测试套件实现，包含15项测试：
1. Frequency (Monobit) Test - 单比特频数检测
2. Frequency Test within a Block - 块内频数检测
3. Runs Test - 游程检测
4. Test for Longest Run of Ones - 块内最长游程检测
5. Binary Matrix Rank Test - 二元矩阵秩检测
6. Discrete Fourier Transform Test - 离散傅里叶检测
7. Non-overlapping Template Matching Test - 非重叠模板匹配检测
8. Overlapping Template Matching Test - 重叠模板匹配检测
9. Maurer's Universal Statistical Test - Maurer通用统计检测
10. Linear Complexity Test - 线性复杂度检测
11. Serial Test - 序列检测
12. Approximate Entropy Test - 近似熵检测
13. Cumulative Sums Test - 累加和检测
14. Random Excursions Test - 随机游动检测
15. Random Excursions Variant Test - 随机游动变体检测

#### `core/gbt_tests.py`
GB/T 32915-2016 补充测试实现，包含3项测试：
1. Poker Test - 扑克检测
2. Autocorrelation Test - 自相关检测
3. Binary Derivative Test - 二元推导检测

#### `core/statistics.py`
统计分析函数，提供：
- 通过率计算
- 均匀性检验
- 置信区间计算
- 汇总统计

### 工具模块

#### `utils/data_loader.py`
数据加载器，支持：
- 多种文件格式（.bin, .txt, .dat, .raw）
- 目录批量加载
- 序列验证和信息获取
- 序列分割

#### `utils/report_generator.py`
报告生成器，支持：
- 文本报告（.txt）
- HTML报告（.html，含图表）
- JSON结果（.json）
- 图表生成

### 配置文件

#### `config.py`
配置文件，包含：
- 显著性水平设置
- 测试参数配置
- 通过标准设置
- 报告配置

## 扩展说明

### 添加新测试

如需添加新的随机性测试：

1. 在 `core/` 目录创建新的测试模块
2. 实现测试函数，返回P-value
3. 在 `main.py` 中集成新测试
4. 更新报告生成器以包含新测试结果

### 修改配置

编辑 `config.py` 文件可以修改：
- 显著性水平（默认0.01）
- 测试参数（块大小、模板长度等）
- 通过标准（通过率阈值）
- 报告格式选项

### 自定义报告

修改 `utils/report_generator.py` 可以：
- 自定义报告格式
- 添加新的图表类型
- 修改报告样式
- 添加新的输出格式

## 依赖说明

### Python版本
- 推荐 Python 3.7+
- 支持 Python 3.6+（部分功能可能受限）

### 依赖包
- numpy >= 1.21.0：数值计算
- scipy >= 1.7.0：科学计算和统计函数
- matplotlib >= 3.5.0：图表生成

### 安装依赖
```bash
pip install -r requirements.txt
```

或使用安装脚本：
```bash
install.bat
```

## 性能说明

### 内存使用
- 每个1Mbit序列约需1MB内存
- 批量处理时建议每批不超过100个文件
- 可使用生成器模式减少内存占用

### 处理时间
- 单个1Mbit序列：约1-5秒
- 1000个序列：约30分钟-2小时
- 取决于CPU性能和测试项数量

### 优化建议
- 使用numpy数组提高计算效率
- 批量处理时使用多进程
- 对于大数据集，使用生成器模式

## 故障排除

### 常见问题

1. **ImportError: No module named 'xxx'**
   - 解决：安装依赖 `pip install -r requirements.txt`

2. **MemoryError**
   - 解决：减少同时处理的文件数量
   - 使用生成器模式加载数据

3. **测试结果异常**
   - 检查数据格式是否正确
   - 验证数据长度是否足够
   - 确认数据是否损坏

4. **图表显示问题**
   - 安装中文字体支持
   - 检查matplotlib配置
